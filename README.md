## 🎨 File Formats
- **.SVG:** Scalable Vector Graphics, perfect for direct editing in Canva, Adobe Illustrator, or other vector graphic tools without losing quality.
- **.ZIP:** Bundled resources containing comprehensive template assets (Note: Some of these files are large, as they contain high-quality assets).

## 💡 How to Use
1. Download the specific `.svg` or `.zip` file of the template you like.
2. If it's a `.zip`, extract the contents to your local machine.
3. Import the files directly into Canva or your preferred design software.
4. Customize the text, colors, and elements to fit your specific event!
